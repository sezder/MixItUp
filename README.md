# Mix It Up

## Summary
[Mix It Up](https://sezder-mix-it-up.herokuapp.com/) is a cocktail and bar discovery and review site. 

#### Core features: 
- Create an account
- Log in to your account
- Log in as a demo user
- Create, edit, update, and delete:
   - Cocktails
   - Reviews
   - Bars
   
   
   
## Technologies Used
Frontend: 
- React
- Redux
Backend: 
- Express
- PostgreSQL
